Elementos Basicos do HTML

HTML é um idioma de marcação que usa elementos para organizar e apresentar o conteúdo da página. Alguns dos elementos mais básicos são:

Clique para expandir
  • <html>: define todo o documento HTML. É o elemento raiz de todos os outros.
  • <head>: é um elemento que contém metadados sobre a página, como o título e links de estilo.
  • <title>: define o título da página que aparece na guia do navegador.
  • <body>: é o elemento que contém todo o conteúdo visível da página.
  • <h1> a <h6>: define cabeçalhos de níveis diferentes. O nível é determinado pelo número de # antes do elemento, onde <h1> é o maior nível e <h6> é o menor.
  • <p>: define um parágrafo.
  • <a>: define um link. O atributo href define o endereço a que o link leva.
